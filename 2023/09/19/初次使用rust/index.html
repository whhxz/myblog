使用 cargo 创建项目，cargo 属于 rust 构建和包管理工具。

–vcs=none 表示创建时不要版本管理

成品如下：

Cargo.toml

cargo 相关命令：

cargo build：编译项目，编译后 exe 在target/debug目录下；如果在 toml 里面添加依赖后，也可以通过该命令下载依赖包。
cargo run：运行项目。
cargo doc --open：查看文档。
